Calendar Plugin Demo

The Calendar plugin allows you to manipulate events in the users calendar.

The basic create, find and delete functions are available on both Android and iOS, but more advanced features are platform specific for now.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.
